# LaserCrafter87
![LaserCrafter87](https://lasercrafter.github.io/images/Screenshot 2023-01-01 222025.PNG)
![](https://lasercrafter.github.io/images/Screenshot 2023-01-01 222045.PNG)

LaserCrafter87 is a TKL PCB for custom TKL keyboards that use the standard a.87 spacing as found in most custom TKL keyboards, but unlike the a.87 standard, this PCB has its USB connector on the bottom. This merely started off as a replacement pcb for the Duck Orion v2.5 and the Duck Unicorn X Legend. This PCB features Tsangan (7u) and normal 6.25u bottom row support with north facing spacebar stabilizer wire orientation, Caps lock and scroll lock indicator LED support, RGB underglow, and support for Alps and MX switches. This is meant to be a more versatile pcb and features will get added as I get good at PCB designing.
