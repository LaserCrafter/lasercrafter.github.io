# LaserCrafter87
![LaserCrafter87](https://lasercrafter.github.io/images/Capture.PNG)

LaserCrafter87 is a TKL PCB for custom TKL keyboards that use the a.87 spacing. This PCB features 10u spacebar support along with a Tsangan and normal 6.25u bottom row support with south facing spacebar stabilizer wire orientation, Caps lock and scroll lock indicator LED support, and top mounted USB-C.
